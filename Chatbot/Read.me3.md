# 🤖 Basic Rule-Based Chatbot

## 📌 Description

This is a simple rule-based chatbot created using Python.
It responds to basic user inputs like greetings and simple questions using predefined replies.

---

## 🎯 Goal

To build a beginner-friendly chatbot using:

* if-elif conditions
* functions
* loops
* basic input/output

---

## ⚙️ Features

* Responds to simple inputs:

  * "hello" → Hi!
  * "how are you" → I'm fine, thanks!
  * "bye" → Goodbye!
* Runs continuously until the user exits
* Case-insensitive input handling

---

## 🛠️ Technologies Used

* Python (no external libraries required)

---

## ▶️ How to Run

1. Install Python on your system
2. Save the program as `chatbot.py`
3. Open terminal or command prompt
4. Run the file:

```bash
python chatbot.py
```

---

## 💬 Example Interaction

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
