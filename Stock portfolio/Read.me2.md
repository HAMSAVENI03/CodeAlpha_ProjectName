# 📈 Stock Portfolio Tracker

## 📌 Description

This is a simple Python program that allows users to track their stock investments.
Users can enter stock names and quantities, and the program calculates the total investment value using predefined stock prices.

---

## 🎯 Features

* User can input stock names and quantities
* Uses a predefined dictionary for stock prices
* Calculates total investment value
* Option to save the result in a `.txt` file

---

## 🛠️ Technologies Used

* Python
* Basic concepts:

  * Dictionary
  * Input/Output
  * Loops
  * Conditional statements
  * File handling

---

## ▶️ How to Run the Program

1. Make sure Python is installed on your system
2. Save the program as `stock_tracker.py`
3. Open terminal or command prompt
4. Run the program using:

   ```
   python stock_tracker.py
   ```

---

## 💻 Example Usage

```
Available stocks: ['AAPL', 'TSLA', 'GOOG', 'MSFT']

Enter stock name: AAPL  
Enter quantity: 2  
Investment in AAPL: 360  

Enter stock name: TSLA  
Enter quantity: 1  
Investment in TSLA: 250  

Enter stock name: done  

Total Investment Value: 610  
```

---

## 📂 Output File

If the user chooses to save the result, a file named `portfolio.txt` will be created containing:

```
Total Investment: 610
```
