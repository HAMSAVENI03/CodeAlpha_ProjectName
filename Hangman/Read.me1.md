# 🎮 Hangman Game (Python)

## 📌 Description

This is a simple text-based Hangman game built using Python.
The player has to guess a hidden word one letter at a time.
The player gets **6 chances** to guess wrong letters before losing the game.

---

## 🎯 Features

* Uses a list of **5 predefined words**
* Random word selection
* User inputs one letter at a time
* Displays correct guesses and blanks
* Limits incorrect guesses to **6 attempts**
* Win or lose message at the end

---

## 🛠️ Concepts Used

* `random` module
* `while` loop
* `if-else` conditions
* Lists and strings
* User input/output

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Copy the Hangman code into a file named:

   ```
   hangman.py
   ```
3. Open terminal or command prompt
4. Run the program:

   ```
   python hangman.py
   ```

---

## 🎮 How to Play

* The game will display blanks (_ _ _ _)
* Enter one letter at a time
* If correct → letter is revealed
* If wrong → attempts decrease
* You have only **6 wrong attempts**
